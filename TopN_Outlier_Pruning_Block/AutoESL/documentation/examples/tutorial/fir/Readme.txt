This tutorial provides an introduction to using AutoESL. 
Using this FIR filter, the tutorial explains how to:

	o Create an AutoESL project
	o Perform validation of the C design
	o Perform Synthesis & Design Analysis
	o Address Bit-accurate design
	o Perform Design Optimization
	o Understand how to perform RTL verification and implementation
	o Review using AutoESL with Tcl scripts

The steps to accomplish these goals using this example are fully documented in the 
AutoESL_Tutorial_Introduction.pdf found in the docs directory in the AutoESL installation area.
